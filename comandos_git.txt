comandos de git :

git init : para iniciar en el repositorio git
git clone: para copiar un repositorio de git
git status: para ver el status
git add . : para agregar todos los archivos -- git add <archivo> para ser especifica
git commit -m " " : para hacer el commit 
git pull : para actualizar el repositorio
git push <origin master> : para subir al repositorio
git push <> --tags : para subir al repo con las tags que hice 
git tag : para ver estiquetas
gir tag -a (v1.0.0) -m "version 1.0.0" (el num del commit) : para crear la tag
git diff <el archivo> : para ver las diferencias
git show <el archivo>: para mostrar un archivo
touch : para crear un archivo
mkdir: para crear una carpeta
git checkout <la rama> : Para cambar de rama
git branch : para ver las ramas
git branch <nombre rama> : para crear una rama
git log --oneline
git merge <la rama que quiero unir> : para unir ramas
git tag -d <la version> : eliminar una version en el local, luego pushear
git rm <archivo> : elimino de forma remota y local
